Goal App API

The resulting API data is formatted as a JSON array. Pre-formatted HTML output is available in addition to raw individual values.

  • q = quote text
  • a = author name
  • c = character count
  • h = pre-formatted HTML quote

Example:

[{
    "q": "Values are like fingerprints. Nobody's are the same, but you leave 'em all over everything you do.",
    "a": "Elvis Presley",
    "c": "98",
    "h": "<blockquote>“Values are like fingerprints. Nobody's are the same, but you 
leave 'em all over everything you do.” — <footer>Elvis Presley</footer></blockquote>"
},
// ...MORE DATA... //
{
    "q": "Nothing is too high for a man to reach, but he must climb with care and confidence",
    "a": "Hans Christian Andersen",
    "c": "82",
    "h": "<blockquote>“Nothing is too high for a man to reach, but he must climb with 
care and confidence” — <footer>Hans Christian Andersen</footer></blockquote>"
}]

Endpoints

/all

Returns all stores objects. Currently there are 300 objects in the quote bank

/quote

Returns one quote object randomly.
